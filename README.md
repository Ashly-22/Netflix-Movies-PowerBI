# Netflix-Movies-PowerBI
Interactive Power BI dashboard built using a Netflix movies dataset. This project includes data cleaning with Power Query, DAX measures, data modeling, and interactive visualizations to analyze movie trends, genres, ratings, and popularity.

## Overview

The dashboard provides insights into:

- Movie release trends over time
- Most popular movies
- Genre distribution
- Average ratings
- Total votes
- Language analysis

## Tools

- Power BI Desktop
- Power Query
- DAX
- Data Modeling

## Features

- Interactive filters
- KPI Cards
- Custom DAX Measures
- Data Transformation
- One-to-Many Relationships
- Professional Dashboard Design

## Dataset

Netflix Movies Dataset
